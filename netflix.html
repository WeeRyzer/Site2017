Netflix Web App

WEB APP • UNI PROJECT

For this project, university asked us to create a web application that would help people find/create information in any area we wanted. We went through three stages, the first was to come up with the idea for the application itself and design through wireframes and mock-ups on Photoshop. The second stage was to create the web app in HTML and CSS and then finally we had to make the application work using API's and JS.

PROJECT INFO

Here is a small preview of how the website works. The user can choose which Netflix region they would like, search for a film/tv series to read all the information on it. They can also find out what is newly added and highest and lowest rated in the Netflix catalogue.

The aim for the application was to provide the user with the most important information first. When you are looking for something to watch you tend to be interested in what has recently been added or what has the highest reviews at that moment. We wanted to give the user the information that they don't really find as much on the actual Netflix website. Maybe the user is looking for a bit of a laugh at the films/programs that aren't as well received from other viewers which is why we added the lowest rated section too. This app could also be implemented for Amazon Prime.
