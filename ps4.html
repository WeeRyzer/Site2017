PS4 UI

USER INTERFACE • WIP

I decided to have a go at recreating the layout of the PlayStation 4 dashboard. This menu is the first screen the user sees when the console is turned on (apart from the one where you choose which account you're logging in with).

PROJECT INFO

If you’re not sure how the dashboard works — this is where you start up anything you want to do on the console whether it be starting up a party with friends, playing games or buying items off of the PlayStation Store.

All original icons were used for the redesign and the top bar would be controlled using the L1 and R1 buttons on the controller. To get back to the home screen the PlayStation button would be pressed.

Since creating this redesign, PlayStation have added a sub-menu which allows the user to find some of the regularly used sections easier.
